		# 연산자(operator)
```
  		- 자바에서 여러 종류의 연산을 수행하기 위한 다양한 기호
  		
  		연산자 종류
  		
  		1. 산술연산자
  		2. 대입연산자
  		3. 증감연산자
  		4. 비교연산자
  		5. 논리연산자
  		6. 비트연산자
  		7. 삼항연산자
```
  		
  		## 산술연산자란? 
```
		- 사칙연산을 다루는 연산자

		

		+ : 왼쪽의 피연산자에 오른쪽의 피연산자를 더하는 기호
		-  : 왼쪽의 피연산자에 오른쪽의 피연산자를 빼는 기호
		*  : 왼쪽의 피연산자에 오른쪽의 피연산자를 곱하는 기호
		/  : 왼쪽의 피연산자에 오른쪽의 피연산자를 나누는 기호
		% : 왼쪽의 피연산자에 오른쪽의 피연산자를 나눈 후, 그 나머지 값을 구하는 기호 

```
		## 대입연산자란?
```
		- 대입 연산자는 변수에 값을 대입할때 사용하는 이항 연산자이며,
		  피연산자들의 결합 방향은 오른쪽에서 왼쪽

		대입연산자 기호 :   = 
```

		### 복합 대입 연산자란?
```
		- 대입연산자 - 와 다른 연산자를 경합하여 만든 연산자

		- 복합 대입 연산자 종류
		
		+=  :  왼쪽의 피연산자에 오른쪽의 피연산자를 더한 후, 그 결과값을 왼쪽의 피연산자에 대입하는 연산자 기호
		
		-=  :  왼쪽의 피연산자에 오른쪽의 피연산자를 뺀 후, 그 결과값을 왼쪽의 피연산자에 대입하는 연산자 기호 

		*=  : 왼쪽의 피연산자에 오른쪽의 피연산자를 곱한 후, 그 결과값을 왼쪽의 피연산자에 대입하는 연산자 기호

		/= :  왼쪽의 피연산자에 오른쪽의 피연산자를 나눈 후, 그 결과값을 왼쪽의 피연산자에 대입하는 연산자 기호

		%= : 왼쪽의 피연산자에 오른쪽의 피연산자를 나눈 후, 그 나머지값을 왼쪽의 피연산자에 대입하는 연산자 기호
```
		## 증감 연산자란?
```
		 - 피연사자를 1씩 증가 혹은 감소시킬때 사용하는 연산자
		 - 이 연산자는 피연산자가 단 하나뿐인 단항 연산자

		- 증감연산자 종류

		++ 증가연산자 : 피연산자의 값을 1증가 시키는 연산자
		-- 감소연산자 :  피연산자의 값을 1감소 시키는 연산자

		++x   :  먼저 피연산자의 값을 1증가 시킨후에 해당 연산을 진행
		x++   :  해당 연산을 수행 하고 나서, 피연산자의 값을 1증가 시킴

		--x   : 먼저 피연산자의 값을 1감소 시킨후에 해당 연산을 진행
		x--   :  해당 연산을 수행하고 나서, 피연산자의 값을 1감소 시킴
```

		# 비교연산자란?
```
		- 피연산자 사이의 상대적인 크기를 판단하는 연산자

		- 비교연산자는 왼쪽의 피연산자와 오른쪽의 피연산자를 비교하여
		 어느 쪽이 더 큰지, 작은지, 또는 서로 같은지를 판단하는 연산자

		- 비교연산자는 모두 두개의 피연산자를 가지는 이항 연산자이며,
		 피연산자들의 결합 방향은 왼쪽에서 오른쪽
```
```
		## 비교연산자 종류

==	왼쪽의 피연산자와 오즌쪽의 피연산자가 같으면 true를 반환

!=	왼쪽의 피연산자와 오른쪽의 핀연산자가 같지 않으면 true를 반환하는 연산자

>	왼쪽의 피연산자가 오른쪽의 피연산자보다 크면 true를 반환하는 연산자

>=	왼쪽의 피연산자가 오른쪽의 피연산자보다 크거나 같으면 true를 반환하는 연산자	

```
		# 논리연산자
```

&& : 논리식이 모두 참이면 참을 반환함 (AND 연산자)

|| : 논리식이 하나라도 참이면 참을 반환함(OR 연산자)

! :  논리식의 결과가 참이면 거짓을 , 거짓이면 참을 반환 (NOT 연산자)
```
