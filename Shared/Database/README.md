기억장치 

주기억장치 - RAM

보조기억장치 - HDD, CD-R/W, FDD. SSD

전제 - 모든 프로그램은 주기억장치에서만 실행됨

프로그래밍 

시스템프로그래밍 - OS제작, 수정

응용프로그래밍(Application) - 엑셀 제작, 원가관리 프로그램 등

웹 프로그래밍 (Site, Homepage, internet)- 홈페이지 제작

기본용어

프로그램(Program) : 하드디스크에 들어 있는 상태
프로세스(Process) : 주기억장치에 로드된 상태의 프로그램(실행되고 있는 프로그램)

※프로그래밍 언어 : 프로그램을 작성하기 위한 언어



※프로그래밍 언어 종류

  

   1.기계어 : 컴퓨터 만 이해하고 있는 0,1



   2.어셈블리어 : 0,1 대신 -> ADD, SUB, MOVE 등 니모닉 기호 사용

     단점: 프로그램구조 나 자료구조 표현어려움



    3.고급언어 : Pascal, Basic, C/C++, C#, Java

        장점 : 1.사람이 이해하고 표현하기 쉬움

                2.복잡한 알고리즘,자료구조 처리 가능
