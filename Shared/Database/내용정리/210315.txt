운영체제란?

OS(Operating System) 라고도 함
- 컴퓨터 시스템의 자원(CPU,주기억장치, 보조기억장치 등)들을 효율적으로 관리

운영체제 종류

개인용 운영체제
 일반적인 OS 의 역할에 충실한 운영체제, 개인 및 기업에서 클라이언트에 설치
Windows, Linux, Mac OS

서버용 운영체제
 기본적인 OS의 역할은 그대로 유지, 다수의 사용자에게 서비스를 제공하기 위한 다양한 기능들이 제공, 클라이언트용에 비해 강화된 보안 기능 제공
Linux, Windows Server

모바일 운영체제
 개인용 PC 또는 서버 외에 모바일 기기에 사용되는 운영체제로 일반 운영체제에 비해 용량이 작음

Android, IOS


Windows 란?
 마이크로소프트 사에서 개발한 OS로 데스크탑에서 사용하는 OS 중 가장 높은 점유율을 가지고 있음. 상용 소프트웨어로써 유료로 구매해야 사용가능하며
Windows 10의 경우 이전 버전으로부터의 무료 업그레이드를 제공

Window CLient Version : 개인 또는 기업에서 사용하는 클라이언트용 윈도우즈

Windows Server Version
 Standard Edition, Datacenter Edition 으로 구분
