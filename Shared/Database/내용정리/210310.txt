데이터 정의어 (DDL)

create Table TableName
(
	deptno	 number(2),
	dname	varchar2(14),
	loc		varchar2(13),
	create_date date default sysdate || [default 값 --> 'DD-MON-RR'] '01-JAN-20'
);


------------------------------------------------------------------------------------------------------

<제약조건>

1. Not null - 컬럼에 null값이 들어오는 것을 막아줌
2. Unique - 중복된 값이 들어오는 것을 막아줌 
3. Primary Key(기본키) - not null + unique | 테이블 당 한번 선언 가능함 
4. Foreign key(외래키) - 특정 테이블의 특정 컬럼을 참조함 || ★ 참조할 테이블에 PK가 걸려있어야만 사용가능
5. Check - 해당 컬럼이 만족해야하는 조건문을 지정함

------------------------------------------------------------------------------------------------------
테이블 절단 (truncate table)
- 테이블 구조는 남겨두고 테이블의 모든 행 제거
- DDL 구문이므로 rollback 안됨

------------------------------------------------------------------------------------------------------

테이블로부터 데이터 검색

SELECT * FROM TableName

여러 테이블로부터 데이터를 검색하고 싶을 때 JOIN ~ ON 사용 [ 동일한 컬럼명의 경우 Table1.colName1 Table2.colName2 로 표시한다]
